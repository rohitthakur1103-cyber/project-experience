Coaching Decision Research Dashboard

This project turns coaching data into a multi-page executive dashboard for retention, transition, hiring, and comparison decisions. The emphasis is front-office decision support rather than match-day analysis.

  • Browser dashboard
  • Decision support
  • Data pipeline
  • Sports operations
Open live app Open repository

Core idea

Coaching decisions are often made across scattered notes, records, and impressions. This dashboard is an attempt to present the relevant information in one place so decision-makers can compare coaches and assess fit with more structure.

Project structure

  • A generated browser dataset powers the dashboard directly.
  • Shared JavaScript handles rankings, compare flows, and report logic.
  • Static pages keep the project easy to host and review.

Coach board

Ranked coach board and dossier page for reviewing profiles in one place.

Search and compare

Natural-language search and side-by-side comparison workflow for decision support.

Patterns

Pattern and explanation page for surfacing higher-level takeaways from the dataset.

Executive brief flow

Browser-based decision framing that keeps the output close to the review workflow.

What this shows: translation of raw sports data into a front-office product, with attention to structure, comparability, and executive readability.
