Job Agent

Job Agent is a local workflow for job discovery, filtering, shortlist generation, and application tracking. The goal is to reduce friction between finding good roles and preparing a focused application packet.

  • Python
  • SQLite
  • CLI + dashboard
  • Application workflow
Open sample packet summary Open sample cover letter Open sample resume variant

Current scope

  • Discovers jobs from Built In and TeamWork Online.
  • Normalizes postings into one schema.
  • Stores jobs and shortlist state in SQLite.
  • Filters for role, location, and salary fit.
  • Supports a local dashboard for application review.

Why it matters

The project is focused on making the application process more systematic. Instead of treating job search as a collection of tabs and notes, it turns it into a repeatable workflow with persistence, filtering, and document preparation.

How it is structured

  • Adapters handle discovery and parsing from multiple sources.
  • Pipeline services manage filtering, scoring, and shortlist generation.
  • SQLite keeps local state for jobs, stages, and generated documents.
  • A dashboard layer surfaces the working queue for review and follow-through.

Public access note

The full interactive app is still a local build. This project page exists so reviewers can still see the workflow, sample outputs, and the type of system that has been built so far.
